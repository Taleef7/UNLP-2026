# Data

## Included

- `dev_questions.csv` -- Development set with 461 questions (2 domains, 41 documents)
- `splits/` -- Cross-validation folds and lockbox partition (generated by `scripts/00_create_splits.py`)

## Not Included

The raw PDF documents (`raw_pdfs/`) are competition data and are not redistributed here. To obtain them, visit the [UNLP 2026 shared task page](https://unlp.org.ua/shared-task/) and follow the data access instructions.

Place the PDFs in `data/raw_pdfs/` organized by domain subdirectory to match the expected directory structure.
